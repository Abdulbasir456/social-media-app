# Social Media App

A full-stack social media application built with:

- **Frontend**: HTML, CSS, JavaScript (Vanilla)
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: JWT, bcrypt

## Project Status

Currently in development (working on core features and layout)
